# Tacan

A Catan board generator built with SvelteKit.

## Developing

```bash
# Install dependencies
pnpm install

# Run dev server
pnpm run dev
```

## Building

### Local

```bash
pnpm run build
```

### Docker

```
docker build --tag tacan .
```
